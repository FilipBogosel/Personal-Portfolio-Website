Projects

Here are some of the projects I've worked on:

  • Events Tracker

    A C++ application that allows users to track events and their details. It includes features for adding, editing, and deleting events, as well as searching for specific events. It has an admin side and a user side with login. The admin part has an intuitive CRUD interface for managing the entire events list. The user side allows the current user to have their own list of events, go through the sorted events list, choose a month for sorting, and view elements displayed as a table to see all the information. There is also the option to open their list in Excel or in a browser, with an HTML file being created for that. I learned very strong OOP concepts building this app. It also has a nice graphical interface built using the Qt framework.

    Technologies: C++, Qt, OOP, STL, Advanced File I/O.

    GitHub Repository

  • Smart-Image-Sorter

    An AI-powered photo organization tool that automatically sorts images into categories like portraits, couples, groups, pets, nature, food, art, vehicles, and urban scenes into folders it will create in the provided folder. By building this project, I learned how to use pre-trained models for image classification and object detection, as well as how to process images using Python Image Library (PIL) and manage files with the Python OS module. It has a PowerShell script that makes all the downloads and runs the program automatically, so you don't have to do anything manually.

    Technologies: Python, PyTorch, torchvision, TensorFlow, ResNet18 (pre-trained on ImageNet) for image classification, ImageAI with RetinaNet for object detection, PIL for image processing, CUDA for GPU acceleration, Python OS library.

    GitHub Repository

  • Battleships Game

    This Battleships game implementation is a strategic naval combat game that allows players to battle against an intelligent AI opponent through both console and GUI interfaces. The project demonstrates advanced object-oriented programming principles with clean modular architecture, featuring separate classes for game logic, board management, and ship placement. Key learning achievements include developing an algorithm that learns from successful hits and strategically targets adjacent cells, implementing comprehensive error handling with custom exceptions for invalid moves, and creating a dual interface design that supports both terminal-based and graphical gameplay. The implementation emphasizes software engineering best practices with 100% test coverage through 25+ unit tests, robust input validation, and maintainable code structure that separates game logic from presentation layers.

    Technologies: Python, PyQt5, OOP, Texttable, Custom Exception Handling, Event-Driven Programming.
